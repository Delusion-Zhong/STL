# 链表
```
typedef struct student
    { 
        // 在这里一般定义节点中的data数据 
        string name; 
        long num;
        int score;
    } ELemType;

    typedef struct Lnode
    {
        // 定义的是数据域和节点域
        ELemType data;   // student 类型的数据域
        struct Lnode *next;  //节点域
    } Lnode, *LinkList;

```
在一般来说指向**头节点就代表整个链表** 使用  ***LinkList** 来定义
```
LinkList L; 
```
